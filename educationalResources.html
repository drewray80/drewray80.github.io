Colabs

What are Colabs?

Colabs are Jupyter Notebooks that will run Python on a Google server through a browser. This allows us to work on colabs from any device with internet. Including school Chromebooks.

Colabs include text blocks that can be used for notes and code blocks that allow students to try concepts. This makes colabs great for scaffolded notes. Creates truly interactive notes.

Even free tier colabs has access to some — Google GPUs and TPUs — that are capable of running AI models. Great for learning, experimenting and developing with AI.

Micro:bits

github.com/drewray80/microbits

As a high school computer science teacher, I really enjoy using microbits in my classes. They offer lots of opportunities for students to learn programming using physical computing.

Microbits have buttons, tons of sensors, leds, a small speaker, radio transmitter and receiver, accelerometer... They can also be used as a microcontroller for other devices. In my classes, we use them to control Sphero RVR+ robots. They are a lot of fun and have many possibilities for students of all ages and ability levels.

The github repo in the link above was created to share how I use microbits in my courses to other teachers, in case the information is useful to someone else.

Linux/Cyber Unit

github.com/drewray80/linux_cyber_unit

In my intro to programming course, I have a brief unit on Linux and cyber security. Experience with linux and using the terminal is a very important base skill for getting started with cyber security, since most cyber security work happens in linux servers. "Learning linux" is challenging without a project. I start this unit by having students use a Raspberry Pi to create a *very* simple website and local serve that site on a LAN. This project provides lots of opportunities to discuss the terminal, servers and clients, IPv4 vs IPv6, and other topics that are key base knowledge for cyber security.

After the website/server project, my students complete an activity with Terminus. Terminus is a text based game that was created by MIT students to teach people about using a linux terminal. It has a retro look. It is challenging and educational. And the students really enjoy it. I found Terminus in a Cyber.org lesson. Cyber.org provides great materials for teachers and teachers can request a free account.

We finish the unit by working through some of the modules in TryHackMe.com. TryHackMe is a great resource for learning about cyber security. The instruction is understandable. The activities are realistic using virtual machines to run linux and windows machines. TryHackMe does have a cost due to the high quality of the materials and activities. I currently use it to introduce cyber to my students, so we just use a few of the free modules. But, if I teach a cyber course in the future, I would definitely try to find funding for the paid version.
