About Me

My Career Now

I am a high school computer science teacher. I teach basic introduction CS courses using Python as the primary programming language. However, I try to present concepts in a general format that will be transferable to any programming languages.

My primary goal is to give students the opportunity to learn useful skills in an engaging environment. I hope that my courses will encourage students who are interested in CS to be motivated to continue in the field. For the students who are not interested in a career in CS, I hope they enjoy the course, and walk away with a basic understanding of CS that they could use in the future.

My Career Experience

In addition to teaching, I held several industry positions, such as a bank manager and an energy conservation consultant. I use those experiences to help guide my instruction to be applicable for my students' future careers.
As a banker, I dealt with a lot of businesses and I also have and MBA. I use this business experience to help students think about their projects from a business prospective. And I help them to think about how to monetize their current and future software projects.
As an energy conservation consultant, I spent a lot of time optimizing HVAC systems to conserve energy. I ust this experience as example of programming uses that students may have not considered. And these physical applications of software are relatable and understandable for all students due to their prior experience.

About this Site

Purpose of this site

Like many (probably most) K12 computer science teachers, I did not have a background in CS. For the last 3 years, I have been learning CS concepts and testing various instructional resources.

I have added many of the resources that I found and developed to my GitHub repositories to help myself to organize my materials and to make them available for anyone that wants them. GitHub can be very challenging to learn (I know from experience), and some of my projects can't be explained well in a github repo. I created this site to help guide people through my resources and give additional context to the code.

Why I wanted to build it

Computer science has become a fascination for me. I have enjoyed learning new aspects about CS to then share with my students. I naturally like to build things, and creating software has been very enjoyable for me. When I discovered GitHub pages, I decided this would be a great opportunity to build a modern website from scratch. So I built this site from scratch using basic html, javascript and tailwindcss.

GitHub

What is GitHub?

GitHub provides repository storage for code. It is the primary location for storing all open source code. It also provides version control for programming, so even code that is not open source is stored on GitHub. It is such and important part of software development that MicroSoft purchased GitHub in 2018 to help insure that it will be maintained. GitHub is not the only option for these services, but it is by far the biggest in the industry.

Due to the processed that GitHub provides, it can be difficult for beginners to understand. As I mentioned in my About this Site section, helping people understand my GitHub repos was a major reason for me to build this site. I also try to use GitHub whenever possible in my CS courses to help students gain experience. Having experience with GitHub is an important skill in the software development market.
